## 📋 Описание задачи

Регулярно возникает задача создания экспертного контента для клиентов и социальных сетей по теме **трудовых прав граждан 2026**.

Контент охватывает:
- ТК РФ 2025-2026 — анализ поправок
- МРОТ и пособия — актуальные суммы от СФР и ФСС
- Судебная практика — типичные споры работников
- Просветительский контент — объяснение прав гражданам
- Статистика Роструда — данные о нарушениях ТК РФ

### Проблема
Без ИИ полный цикл создания контент-пакета занимает **16-24 часа** ручной работы.
